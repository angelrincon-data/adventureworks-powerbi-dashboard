# 📊 AdventureWorks Sales Analytics Dashboard

## 🧠 Business Problem

The management team needs a way to track key business metrics such as sales, revenue, profit, and returns, while also analyzing customer behavior and product performance across regions.

---

## 🎯 Objective

Develop an end-to-end Business Intelligence solution to:

- Monitor KPIs (Sales, Revenue, Profit, Returns)
- Analyze product and regional performance
- Identify high-value customers
- Enable data-driven decision-making

---

## 🛠️ Tools & Technologies

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query (ETL)
- Data Modeling (Star Schema)

---

## 🔄 Data Workflow

1. **Data Extraction**
   - Imported raw CSV files (Sales, Products, Customers, Returns)

2. **Data Transformation (ETL)**
   - Cleaned and transformed data using Power Query
   - Handled missing values and data inconsistencies
   - Created calculated columns

3. **Data Modeling**
   - Built a relational data model (Star Schema)
   - Defined relationships between fact and dimension tables

4. **DAX Calculations**
   - Created measures for:
     - Total Sales
     - Profit
     - Return Rate
     - KPI comparisons

5. **Data Visualization**
   - Designed interactive dashboard with filters and slicers
   - Enabled drill-down analysis by region, product, and customer

---

## 📊 Key Insights

- Sales performance varies significantly across regions
- A small percentage of customers generates a large portion of revenue
- Certain product categories have higher return rates
- Seasonal trends impact sales performance
